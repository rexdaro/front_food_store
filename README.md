# 🎨 FoodStore Frontend

Este es el cliente del sistema FoodStore, desarrollado con React, TypeScript y Tailwind CSS.

## 🚀 Inicio Rápido

1. Instalación: `npm install`
2. Desarrollo: `npm run dev`

## 🏗️ Arquitectura
Utiliza **Feature-Based Architecture**. Cada módulo dentro de `src/features` contiene sus propios hooks, componentes, servicios y tipos.

## 🔑 Tecnologías Clave
- **TanStack Query**: Para el manejo de caché y estados asíncronos de la API.
- **Tailwind CSS**: Para una interfaz moderna y responsive.
- **Lucide React**: Set de iconos premium.
